# AgniQ - Quantum Computing Experience

A progressive cyberpunk-styled interactive journey through quantum computing concepts, built with TypeScript and modern web technologies.

## Overview

AgniQ is an immersive educational platform that takes users through a step-by-step exploration of quantum computing fundamentals. From basic quantum mechanics to advanced thermodynamics, each step builds upon the previous one with interactive visualizations and real-time simulations.

## Features

### Progressive Learning Path

- **Step 1**: Introduction to Quantum Computing
- **Step 2**: Quantum Basics and Superposition
- **Step 3**: Interactive Bloch Sphere Visualization
- **Step 4**: Measurement and Wave Function Collapse
- **Step 5**: Quantum Entanglement Demonstrations
- **Step 6**: Quantum Gates and Circuit Building
- **Step 7**: Quantum Thermodynamics

### Interactive Components

- Real-time quantum state visualizations
- 3D Bloch sphere manipulations
- Particle system animations
- Matrix rain effects and cyberpunk aesthetics
- Integrated testing mode for step navigation
- Thermodynamic framework with heat engine simulations

### Technical Features

- TypeScript-based architecture
- Modular step management system
- Canvas-based visualizations
- Responsive design with Tailwind CSS
- Performance monitoring and optimization
- Audio-visual effects and glitch animations

## Quick Start

### Prerequisites

- Node.js (v14 or higher)
- Modern web browser with ES2020 support

### Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/DevAgnihotri/AgniQ.git
   cd AgniQ
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the project**

   ```bash
   # Standard launch (port 3001)
   ./startup.bat

   # Auto port detection (3001-3010)
   ./startup-auto.bat
   ```

4. **Open in browser**
   Navigate to `http://localhost:3001` (or the auto-detected port)
